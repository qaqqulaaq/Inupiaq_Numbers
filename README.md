# Inupiaq Numbers
This program is a simple practice tool for studying vocabulary for numbers in IÃ±upiaq, as well as the Kaktovik numeral system. It currently
has 5 modes: 

-Number Operations for practicing addition and subtraction with Kaktovik Numerals
-Number Conversions for practicing converting between Arabic and Kaktovik numerals
-Number to Word for matching vocabulary with the numerals
-Vocabulary Multichoice as a simple multiple choice exercise to practice vocabulary for numbers
-Vocabulary Wordbuild for practicing the arrangement of number bases and postbases

Future planned expansions include multiplication and division, fractions, and practical uses for words for numbers.
